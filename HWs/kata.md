# Kata (Eng)

## All Stuff, No Cruft

### Description

Conference organizer needs a management system for the conferences he runs

### Requirements

* attendees can access speaking schedule online, including room assignments;
* speakers can manage talks (enter, edit, modify);
* attendees "vote up/down" talks
* organizer can notify attendees of schedule changes up-to-the-minute (if attendees opt in)
* each conference (being a different subject) can be branded independently
* speaker slides are accessible online only to attendees
* evaluation system via web page, email, SMS, or phone

### Users

* hundreds of speakers
* dozens of event staff
* thousands of attendees

# Kata (RUS)

## All Stuff, No Cruft («Все нужное — ничего лишнего»)

### Описание

Организатор конференций нуждается в системе управления для проводимых им мероприятий.

### Требования

* Участники могут просматривать расписание выступлений онлайн, включая распределение по залам.
* Докладчики могут управлять своими выступлениями (создавать, редактировать, изменять).
* Участники могут голосовать за выступления («за»/«против»).
* Организатор может уведомлять участников об изменениях в расписании в режиме реального времени (если участники подписались на уведомления).
* Каждая конференция (с разной тематикой) может иметь собственный уникальный бренд.
* Слайды докладчиков доступны онлайн только для участников.
* Система оценки доступна через веб-страницу, электронную почту, SMS или телефон.

### Пользователи

* сотни докладчиков,
* десятки сотрудников мероприятия,
* тысячи участников.
